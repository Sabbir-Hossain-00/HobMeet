# Project Name: HobMeet

## Overview
HobMeet is a social platform built with the MERN stack that helps hobbyists connect, share, and grow their communities. Whether your interest lies in music, coding, painting, or reading, HobMeet brings people together through shared passions with a clean, modern interface.

## Main Technologies
- React.js  
- Tailwind CSS  
- Firebase  
- Express.js  
- MongoDB  

## Key Features
- Discover and join hobby-based communities  
- Connect with like-minded people  
- Create and participate in hobby events and meetups  
- Responsive and modern UI/UX with animations  
- Secure authentication using Firebase  
- Full-stack functionality with Node.js and Express

## How to Run Locally
1. Clone the repository:  
   `git clone https://github.com/yourusername/hobmeet.git`

2. Navigate into the project directory:  
   `cd hobmeet`

3. Install dependencies:  
   `npm install`

4. Start the development server:  
   `npm start`

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
